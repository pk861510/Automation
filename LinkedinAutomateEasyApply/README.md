# 🚀 LinkedIn Easy Apply Automation Bot (Python + Selenium)

Automated LinkedIn Easy Apply bot built using Python and Selenium that intelligently applies to jobs, handles multi-step forms, pagination, dropdowns, and typeahead fields like location.

This project demonstrates real-world browser automation, workflow optimization, and intelligent form handling — similar to automation tools used in real companies to improve productivity.

---

# 🎯 Project Objective

Help job seekers automate repetitive LinkedIn Easy Apply workflows and improve efficiency by:

- Automatically applying to Easy Apply jobs
- Handling multi-step job application forms
- Filling numeric, dropdown, and typeahead fields intelligently
- Navigating across multiple job pages using pagination
- Preventing background tab issues
- Reducing manual effort and saving time


# ⚡ Key Features

## 🤖 Fully Automated Job Application
- Automatically detects Easy Apply jobs
- Opens job cards safely (prevents new tabs)
- Applies to multiple jobs automatically
- Supports large-scale job automation

## 🧠 Intelligent Form Filling

Handles all LinkedIn form types:

- Experience fields
- Expected salary fields
- Notice period fields
- Email and phone number
- Location field (auto fills Pune)
- Dropdown fields
- Radio buttons (Yes / No)
- Multi-step applications

## 📄 Pagination Support

- Automatically clicks Next button
- Loads all job pages
- Applies across multiple pages automatically

## 🔒 Safe Automation Features

- Prevents new tab opening
- Handles stale elements safely
- Uses real LinkedIn session
- Avoids crashes and failures
- Skips already applied jobs safely

---

# 🧠 Automation Workflow

```
LinkedIn Job Search Page
        ↓
Load Job Cards
        ↓
Click Job Card
        ↓
Click Easy Apply
        ↓
Fill Form Automatically
        ↓
Handle Dropdowns & Location Field
        ↓
Submit Application
        ↓
Click Done
        ↓
Move to Next Job
        ↓
Move to Next Page
```
