

# 🚀 Job Application Automation Suite (LinkedIn • Naukri • Indeed)

A collection of **Python + Selenium automation bots** that automatically apply to jobs across major job platforms.

This repository demonstrates **real-world browser automation, workflow automation, and intelligent form handling** used to streamline repetitive job application processes.

---

# 🎯 Project Overview

This project automates job applications on multiple platforms by:

- Automatically detecting **Apply buttons**
- Filling **job application forms**
- Handling **multi-step applications**
- Managing **popups, iframes, and dynamic fields**
- Detecting **captcha challenges**
- Logging application results

The goal is to reduce repetitive manual effort and demonstrate **practical automation engineering skills**.

# ⚡ Supported Platforms

### 🔹 LinkedIn Automation
- Automatically applies to **Easy Apply jobs**
- Handles multi-step application forms
- Supports pagination across job results

### 🔹 Naukri Automation
- Applies only to **Naukri in-site applications**
- Skips external company-site jobs
- Handles modals, popups, and iframes

### 🔹 Indeed Automation
- Detects **Apply on Indeed**
- Automatically fills job application forms
- Handles captcha detection and multi-step forms
